# Инструкция по загрузке аудио файлов

## Формат и требования

- **Формат:** MP3
- **Максимальный размер для рассказов:** 10MB
- **Максимальный размер для слов:** 5MB

## Где сохраняются файлы

### Рассказы
- **Путь:** `storage/app/public/audio/stories/`
- **Имя файла:** `{id_рассказа}.mp3`
- **Пример:** `storage/app/public/audio/stories/1.mp3`

### Слова
- **Путь:** `storage/app/public/audio/words/`
- **Имя файла:** `{id_слова}.mp3`
- **Пример:** `storage/app/public/audio/words/1.mp3`

## Как загрузить аудио

### Для рассказов:
1. Перейдите в админ-панель → Рассказы
2. Создайте новый рассказ или отредактируйте существующий
3. В форме есть поле "Аудио файл (MP3)"
4. Выберите MP3 файл и сохраните
5. Файл автоматически сохранится в нужную директорию

### Для слов:
1. Перейдите в админ-панель → Словарь
2. Создайте новое слово или отредактируйте существующее
3. В форме есть поле "Аудио файл (MP3)"
4. Выберите MP3 файл и сохраните
5. Файл автоматически сохранится в нужную директорию

## Важные замечания

- При загрузке нового файла старый автоматически удаляется
- Можно удалить аудио файл, установив галочку "Удалить текущий аудио файл" при редактировании
- Если аудио файл не загружен, будет использоваться браузерная озвучка (Web Speech API)
- Убедитесь, что директории `storage/app/public/audio/stories/` и `storage/app/public/audio/words/` существуют и доступны для записи

## Рекомендации по качеству

- Используйте качественные MP3 файлы (битрейт 128-192 kbps)
- Убедитесь, что аудио четко произносит японский текст
- Для рассказов рекомендуется использовать один голос для всего текста
