# Инструкция по созданию базы данных для Obake

## Шаг 1: Создание базы данных через phpMyAdmin

1. Откройте phpMyAdmin в браузере (обычно `http://localhost/phpmyadmin`)

2. Нажмите на вкладку **"Базы данных"** (Databases) в верхнем меню

3. В поле **"Создать базу данных"** (Create database) введите имя:
   ```
   obake
   ```

4. Выберите кодировку: **`utf8mb4_unicode_ci`** или **`utf8mb4_general_ci`**

5. Нажмите кнопку **"Создать"** (Create)

База данных будет создана!

## Шаг 2: Настройка подключения в .env

Откройте файл `.env` в корне проекта и убедитесь, что настройки базы данных правильные:

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=obake
DB_USERNAME=root
DB_PASSWORD=
```

**Важно:** 
- `DB_USERNAME` - обычно `root` для локальной разработки
- `DB_PASSWORD` - оставьте пустым, если у root нет пароля, или укажите ваш пароль

## Шаг 3: Выполнение миграций

После создания базы данных выполните миграции, чтобы создать все таблицы:

```bash
php artisan migrate
```

Эта команда создаст все необходимые таблицы:
- users
- stories  
- global_dictionary
- user_dictionary
- story_words
- и другие системные таблицы

## Альтернативный способ: Создание через SQL

Если предпочитаете использовать SQL, выполните в phpMyAdmin:

```sql
CREATE DATABASE obake CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

Затем выберите созданную базу данных и выполните миграции через Laravel.

## Проверка

После выполнения миграций вы должны увидеть все таблицы в базе данных `obake` в phpMyAdmin.
